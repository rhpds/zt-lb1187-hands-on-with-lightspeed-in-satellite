= Red Hat Lightspeed vulnerability service in Red HatSatellite

In this module you will explore the Red Hat Lightspeed vulnerability service in Satellite.

The Red Hat Lightspeed vulnerability service allows users to assess, triage, prioritize and remediate the most critical vulnerabilities affecting their servers with the built-in threat intelligence. The Red Hat Lightspeed vulnerability service is a technical preview feature.

=====
.View the vulnerabilities
[%collapsible]
====

Let's view the vulnerabilities detected on the host registered to the Satellite server.

Click on the `Vulnerability` menu item on the left hand navigation bar.

image::vulnerability-menu.png[link=self, window=blank, width=100%]

You will see a list of vulnerabilities detected on the host registered to the Satellite server.

image::vulnerabilities-list.png[link=self, window=blank, width=100%]

Click on the vulnerability `CVE-2025-5283`.

You will be taken to a page that shows the details of the vulnerability.

image::cve-2025-5283.png[link=self, window=blank, width=100%]

The vulnerability details page shows the details of the vulnerability including the CVSS score, and the description.

You can view the details by clicking on the `View in Red Hat CVE database` link.

image::vulnerability-details.png[link=self, window=blank, width=100%]

The vulnerability service is in technical preview status thus remediation is not available.
